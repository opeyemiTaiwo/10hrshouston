## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build
```

## Deploy to Vercel

```bash
# Install Vercel CLI (if needed)
npm i -g vercel

# Deploy
vercel
```

Or connect the GitHub repo in your [Vercel Dashboard](https://vercel.com/dashboard) for automatic deployments.

- Google Fonts (Oswald + DM Sans)
